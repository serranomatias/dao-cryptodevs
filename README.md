# CryptoDevsDAO

This DAO allows to CryptoDevsNFT Holders to create and vote on proposals. Within the DAO exists a Treasury Balance and the people could vote on the proposal to decide how will be used.

## More Details

### What i use
- Next.js
- Solidity
- Hardhat
- Chai (test)
- Web3Modal
- Ethers.js
- dotenv
